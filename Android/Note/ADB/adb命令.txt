查看连接的设备
adb devices

安装XXX文件，不保留以前在手机中的数据
adb install xxx

安装XXX文件，保留以前在手机中的数据
adb install -r APK_FILE

卸载App
adb uninstall PACKAGE_NAME

启动一个Activity
adb shell am start PACKAGE_NAME/ACTIVITY_IN_PACKAGE
adb shell am start PACKAGE_NAME/FULLY_QUALIFIED_ACTIVITY

进入shell
adb shell

退出shell
exit

模拟按下电源键
adb shell input keyevent 26

模拟解锁屏幕
adb shell input keyevent 82

打印所有安装的app的包
adb shell pm list packages -f

清除app的数据
adb shell pm clear PACKAGE_NAME

查看在命令行log日志
adb logcat

为log日志添加过滤器
adb logcat -s TAG_NAME
adb logcat -s TAG_NAME_1 TAG_NAME_2

为log添加优先级过滤
adb logcat "*:PRIORITY" 
eg：adb logcat "*:W"
V - Verbose (lowest priority)
D - Debug
I - Info
W - Warning
E - Error
F - Fatal
S - Silent (highest priority, on which nothing is ever printed)

清除logcat buffer
adb logcat -c
eg：adb logcat "*:PRIORITY"
adb logcat -s TAG_NAME_1 TAG_NAME_2

深入解读：
1、adb [-d|-e|-s <serialNumber>] <command>
-d代表硬件设备   -e代表模拟器  -s代表硬件serial
eg；adb -s emulator-5556 install helloWorld.apk

2、改变设备端口
adb forward tcp:6100 tcp:7100
将6100的端口改为7100   

3、
adb pull <remote> <local>   从设备获取文件
adb push <local> <remote>   拷贝文件到设备里面去
eg：adb push foo.txt /sdcard/foo.txt

4、在shell里面录屏并提取
$ adb shell
shell@ $ screenrecord --verbose /sdcard/demo.mp4
(press Ctrl-C to stop)
shell@ $ exit
$ adb pull /sdcard/demo.mp4
